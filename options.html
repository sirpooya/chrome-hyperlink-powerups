Select and copy multiple links by dragging your mouse while holding a custom key!

How to use:
  1. Set your custom shortcut key below (e.g., "Z")
  2. Hold down your custom key
  3. Click and drag to create a blue selection box
  4. Release to copy all links in the selected area to clipboard
When enabled, all links with target="_blank", target="_new", and JavaScript window.open() calls will open in the same tab instead of new windows/tabs.

Enter a single letter (A-Z) to use for link selection. Hold this key and drag to select multiple links.
